Lexicon - Vocabulary Learner

A beautiful, interactive React app to learn 100 English words across 10 difficulty levels.

## Features

- **📖 Flashcard Mode**: Flip cards to reveal definitions and etymology
- **⚡ Quiz Mode**: Test yourself with multiple-choice questions
- **🎯 10 Difficulty Levels**: From foundational to lexical mastery
- **🔥 Streak Tracking**: Monitor your learning progress
- **🌱 Etymology**: Learn word roots and origins

## Getting Started

### Prerequisites
- Node.js (download from [nodejs.org](https://nodejs.org))
- Git

### Installation

1. Clone this repository:
```bash
git clone https://github.com/WhatIsSutsDoing/Vocab2.git
cd Vocab2
```

2. Install dependencies:
```bash
npm install
```

3. Run locally:
```bash
npm run dev
```

## Deploying as a public web app

This project is ready to publish as a static site on GitHub Pages.

1. Push the repository to GitHub.
2. Open the repository settings and enable GitHub Pages.
3. Set the source to GitHub Actions.
4. The workflow in [.github/workflows/deploy.yml](.github/workflows/deploy.yml) will build and publish the app automatically on each push to the main branch.

After deployment, the app will be available at:
```text
https://<your-username>.github.io/Vocab2/
```
